三模式对比
维度SSH 直连 ⭐ElasticsearchDuckDB
前置条件SSH 账号密码ES 地址 + 账号无
查询性能2-4s grep 远程<1s 原生聚合<1s SQL
聚合能力Python 内存原生 terms/histogram原生 SQL GROUP BY
数据量万级亿级千万级
基础设施零依赖需 ELK 集群单文件
适用场景快速起步 / 中小团队已有 ELK / 大规模本地开发 / 离线分析
一行配置，三种模式无缝切换
# config.yaml datasource: type: ssh # ← 改为 elasticsearch 或 duckdb 即可
升级路径：SSH（5 分钟上手）→ DuckDB（本地存储）→ Elasticsearch（生产规模）
